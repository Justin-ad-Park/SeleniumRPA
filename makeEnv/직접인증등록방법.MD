# ID, 비밀번호 키체인 저장
### Mac
```
security add-generic-password -a marqvision -s marqvision.login -w '{"email":"your_email@example.com","password":"your_password"}' -U 

# 조회 (Java 없이 수동 확인)
security find-generic-password -a marqvision -s marqvision.login -w
```

## Windows
```
# PowerShell 모듈 한 번 설치:
Install-Module -Name CredentialManager -Scope CurrentUser -Force

# 저장 (Target=marqvision.login, UserName은 식별자, Secret은 JSON 문자열)
$sec = ConvertTo-SecureString '{"email":"your_email@example.com","password":"your_password"}' -AsPlainText -Force
New-StoredCredential -Target 'marqvision.login' -UserName 'marqvision' -Password $sec -Persist LocalMachine

# 조회 (평문 추출)
(Get-StoredCredential -Target 'marqvision.login').GetNetworkCredential().Password
```
